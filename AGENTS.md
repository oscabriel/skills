# Agent instructions

This is a small personal skills repo.

- Keep skills as top-level directories while the repo only has a few skills.
- Each skill must have a `SKILL.md` with valid frontmatter.
- Keep supporting docs next to the skill that uses them.
- Update `README.md` when adding, removing, or renaming a skill.
- Do not add buckets, `CONTEXT.md`, or ADRs unless explicitly requested.
